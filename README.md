# User Authentication System

This project is developed as part of college training using Core Java, JDBC, and MySQL.

## Description
A console-based application that allows users to register and login using database connectivity.

## Features
- User Registration
- User Login
- Role-based Access (Admin/User)

## Technologies Used
- Core Java
- JDBC
- MySQL
- Eclipse IDE

## How to Run
1. Execute `auth_db.sql` in MySQL Workbench
2. Open the project in Eclipse
3. Add MySQL JDBC Driver to Build Path
4. Run `MainApp.java`
